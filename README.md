## Возможности

- 📱 **Сетка приложений** — настраиваемый размер иконок (56/72/96 px), колонок (4–10) и строк (3–8)
- 📄 **Пагинация** — постраничное отображение с индикатором-точками и свайпом
- 📁 **Папки** — создание перетаскиванием, превью 3×3, переименование, удаление
- 🔍 **Поиск** — мгновенный поиск по названию, описанию и ключевым словам (debounce 150ms)
- ✏️ **Переименование** — локальные псевдонимы приложений без влияния на систему
- ⭐ **Закрепление** — избранные приложения всегда первые, перестановка drag-and-drop
- 👁️ **Скрытие** — скрытие приложений из сетки с возможностью восстановления
- 🎨 **Темы** — автоматическая, светлая, тёмная
- ⚡ **Автозакрытие** — при запуске приложения или потере фокуса окна
- 🔑 **Горячая клавиша** — глобальный шорткат с проверкой конфликтов
- 🔄 **Обновления** — автоматическая проверка новых версий через GitHub
- 🖱️ **Drag-and-Drop** — создание папок, добавление в папки, реордеринг закреплённых
- 🎯 **Умные иконки** — автоскругление квадратных иконок, прозрачность для современных
- 🚫 **Фильтр Dock** — скрытие приложений, закреплённых в панели GNOME

## Установка

### Stapler

```bash
stplr repo add adwyra https://github.com/cheviiot/adwyra.git
stplr refresh && stplr install adwyra
```

### Вручную

```bash
git clone https://github.com/cheviiot/adwyra.git
cd adwyra && ./adwyra
```

<details>
<summary><strong>Зависимости</strong></summary>

| Дистрибутив | Команда |

| Debian/Ubuntu | `sudo apt install python3-gi gir1.2-gtk-4.0 gir1.2-adw-1` |
| Fedora | `sudo dnf install python3-gobject gtk4 libadwaita` |
| Arch | `sudo pacman -S python-gobject gtk4 libadwaita` |
| ALT Linux | `sudo apt-get install python3-module-pygobject3 libgtk4 libadwaita` |

</details>

## Использование

```bash
adwyra          # Открыть
adwyra --toggle # Показать/скрыть
adwyra --show   # Показать
adwyra --hide   # Скрыть
```

**Совет:** назначьте `adwyra --toggle` на горячую клавишу в настройках GNOME или прямо в приложении.

## Управление

| Действие | Способ |
|----------|--------|
| Запустить приложение | Клик |
| Поиск | Клик по полю поиска или `Ctrl+F` |
| Создать папку | Перетащить приложение на другое |
| Добавить в папку | Перетащить приложение на папку |
| Переупорядочить закреплённые | Перетащить закреплённое на закреплённое |
| Контекстное меню | ПКМ на приложении/папке |
| Переименовать | ПКМ → Переименовать |
| Закрепить / Открепить | ПКМ → Закрепить |
| Скрыть приложение | ПКМ → Скрыть |
| Настройки | Кнопка ⚙ |
| Закрыть | `Esc` |

## Настройки

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| Колонки | Количество колонок в сетке (4–10) | 7 |
| Строки | Количество строк на странице (3–8) | 5 |
| Размер иконок | Маленький / Средний / Большой | Маленький |
| Тема | Системная / Светлая / Тёмная | Системная |
| Закрывать при запуске | Закрывать окно после запуска приложения | Вкл |
| Закрывать при потере фокуса | Закрывать при клике вне окна | Вкл |
| Скрывать приложения из Dock | Не показывать закреплённые в панели GNOME | Вкл |
| Горячая клавиша | Глобальный шорткат для вызова | — |

## Хранение данных

Все данные хранятся в `~/.config/adwyra/`:

| Файл | Назначение |
|------|-----------|
| `config.json` | Настройки приложения |
| `favorites.json` | Закреплённые приложения |
| `folders.json` | Папки и их содержимое |
| `aliases.json` | Пользовательские имена |
| `hidden.json` | Скрытые приложения |
