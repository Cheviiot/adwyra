## О проекте

Adwyra — компактный лаунчер приложений, написанный на Python с GTK 4 и Libadwaita. Квадратное окно без декораций, сетка с прокруткой, встроенный поиск, папки, закрепление, плагины — всё что нужно для быстрого запуска приложений на рабочем столе GNOME.

## Возможности

- **Сетка приложений** — настраиваемые колонки, строки и размер иконок, постраничная прокрутка свайпом
- **Папки** — создание перетаскиванием одного приложения на другое, превью 3×3
- **Поиск** — по имени, описанию и ключевым словам с debounce
- **Закрепление** — избранные приложения в начале сетки, перестановка drag-and-drop
- **Переименование** — локальные псевдонимы, не затрагивающие систему
- **Скрытие приложений** — убрать из сетки с возможностью вернуть
- **Темы** — системная, светлая, тёмная
- **Горячая клавиша** — глобальный шорткат через GNOME Settings с проверкой конфликтов
- **Автозакрытие** — при запуске приложения или потере фокуса
- **Фильтр Dock** — скрытие приложений, закреплённых в панели GNOME
- **Умные иконки** — автоскругление квадратных иконок по анализу альфа-канала
- **Плагины** — система расширений с API для UI, меню и настроек
- **Проверка обновлений** — через GitHub API

## Установка

### Stapler

```bash
stplr repo add luma https://github.com/Cheviiot/Luma.git
stplr refresh && stplr install adwyra
```

### Из исходников

```bash
git clone https://github.com/cheviiot/adwyra.git
cd adwyra
./adwyra
```

## Использование

```bash
adwyra            # открыть
adwyra --toggle   # показать / скрыть
adwyra --show     # показать
adwyra --hide     # скрыть
```

Назначьте `adwyra --toggle` на горячую клавишу — это можно сделать прямо в настройках приложения.

## Управление

| Действие | Как |
|---|---|
| Запуск приложения | Клик |
| Поиск | Клик по полю или `Ctrl+F` |
| Создать папку | Перетащить приложение на другое |
| Добавить в папку | Перетащить на папку |
| Переставить закреплённые | Перетащить закреплённое на закреплённое |
| Контекстное меню | Правая кнопка мыши |
| Настройки | ⚙ в левом верхнем углу |

## Плагины

Плагины загружаются из двух директорий:

- `/usr/lib/adwyra/plugins/` — системные (установленные через Stapler)
- `~/.config/adwyra/plugins/` — пользовательские

Каждый плагин — папка с `plugin.py`, содержащим класс `Plugin(AdwyraPlugin)`. Плагины могут добавлять пункты контекстного меню, страницы настроек, реагировать на запуск приложений и показ/скрытие окна.

## Данные

`~/.config/adwyra/`

| Файл | Назначение |
|---|---|
| `config.json` | Настройки |
| `favorites.json` | Закреплённые приложения |
| `folders.json` | Папки |
| `aliases.json` | Псевдонимы |
| `hidden.json` | Скрытые приложения |
| `plugins.json` | Состояние плагинов |
| `plugins/` | Пользовательские плагины |
| `plugin_data/` | Данные плагинов |
